WorldCupMockDraw

説明

これは、2026FIFAサッカーワールドカップの抽選会シミュレーションです。

目的

サッカーを盛り上げる。そして、サッカー日本代表を応援する目的で作りました。

経緯

お知り合いの井上マーさんとノーミルク佐藤さんのサッカー解説チャンネルミルアカが切っ掛けで作りました。

作成手法

実験的にスマホ+AIを試みています。

  • 完全スマホのみで作成しました。
  • X（旧Twitter）のAI、Grokにほぼコーディングを任せています。

感想

AIのコーディングの速さは優秀です。プログラマーが数時間掛かるものも数秒で書き上げます。情報収集も早いです。
やけに自信満々で、常に完璧だとアピールしてきます。
しかし、実際は間違えも多いです。
プログラムソースも動かないものを完璧と提示してきますし、データも一見正しそうに見えても信用出来ないです。
例えば最新のFIFAランキングと言いつつ数年前のものを適用します。
→指摘し、何が正しのかを導けば、瞬時に修正してくれます。

結果： 明確なゴールがあり、正しく導くことが出来ればとても優秀です！
